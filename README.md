# LibraryManagerWpf
IMPORTANT: You need to add your ChatGPT API Key to the 'apikey.txt' file.

What does this app?
- save and load data from an json file
- view books in Datagrid
- edit books
- save books
- delete books
- generate random books with ChatGPT API (title, author, description, isbn)
